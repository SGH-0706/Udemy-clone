# Udemy Clone Website

This project is a simple Udemy clone webpage created using HTML and CSS.  
The website is designed as a single-page layout inspired by the Udemy homepage.

## Features

- Responsive Navigation Bar
- Search Bar Section
- Course Categories
- Banner / Hero Section
- Recommended Courses Section
- Topics Recommended Section
- Most Popular Courses Section
- Footer Section

## Technologies Used

- HTML5
- CSS3
- Flexbox Layout

## Concepts Practiced

This project helped in practicing:
- HTML Page Structure
- CSS Styling
- Flexbox Properties
- Responsive Layout Design
- Navigation Bar Design
- Card Layout Creation
- Image and Content Alignment

## Purpose of the Project

The purpose of this project is to improve front-end web development skills by recreating a popular learning platform interface using only HTML and CSS.

## Project Type

- Single Page Website
- Front-End Clone Project
